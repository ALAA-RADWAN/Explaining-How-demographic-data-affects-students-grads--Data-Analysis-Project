<h1 align="center">Student Demographics and Performance Analysis</h1>

<p align="center">
A data analytics project exploring how demographic and educational factors influence student outcomes at the Open University.
</p>

---


## 📌 Project Overview

This project focuses on analysing **studentInfo.csv**, a dataset containing demographic and educational information about Open University students.  
The goal is to uncover patterns that influence student success, withdrawal, and distinction rates.

---

## 🧩 Dataset Description

The dataset includes the following columns:

| Column | Description |
|--------|--------------|
| `code_module` | Identification code for the module |
| `code_presentation` | Code for the presentation period |
| `id_student` | Unique student ID |
| `gender` | Student’s gender |
| `region` | Geographic region of residence |
| `highest_education` | Highest education level on entry |
| `imd_band` | Index of Multiple Deprivation band |
| `age_band` | Age group |
| `num_of_prev_attempts` | Number of previous attempts |
| `studied_credits` | Total credits currently studied |
| `disability` | Disability declaration |
| `final_result` | Final result (Pass, Fail, Withdrawn, Distinction) |

---

## 🎓 Research Questions

### **Question 1:**  
Can demographic factors predict whether a student is likely to withdraw from their course?

> This explores whether age, gender, disability, and deprivation index can serve as early indicators of withdrawal risk.

### **Question 2:**  
Can socioeconomic background (IMD band) affect the student’s final result?

> Investigates whether students from more deprived areas face additional challenges to pass or achieve distinctions.

### **Question 3:**  
Can educational background classify students likely to achieve a distinction?

> Examines whether prior education level correlates with top performance outcomes.

---

## 🧠 Data Analysis Approach

1. **Exploratory Data Analysis (EDA)**  
   - Summary statistics  
   - Missing value checks  
   - Distribution plots for categorical variables  

2. **Visualization**  
   - Pie charts for final results  
   - Bar charts for IMD vs. final result  
   - Heatmaps for correlation between demographic factors
